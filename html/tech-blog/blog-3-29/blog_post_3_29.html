Tech Update from Saturday, 3/29/25

Week Summary


Character Movement

We implemented the basic moveset of our character which includes which includes movement, free camera control, a jump, and a dash. We also spent some time refining the movement with snappier rotation, more control in air, and ensuring the dash is balanced between the air and ground.


Rhythm Manager

One of our major design goals is input on beat, so we worked on a Rhythm Manager which could be called to determine if actions were on beat or not. The Rhythm manager is implemented successfuly using the in game clock as of right now.


Wwise Integration

Since we have a game with an emphasis on audio, our audio programmer intergrated wwise into the project. It was integrated in a manner such that only our audio programmer needs wwise installed and so long as each team member has the unreal wwise plugin, the project compiles fine.
